Ваши основные модули: «Администрирование», «Сервер ТП», «Файловый обмен». Обеспечиваете работоспособность всей инфраструктуры СПРУТ.
🔧 Администрирование — Глава 3
💻
Установка клиентской части BackOffice
›
1
Требования: Windows, Oracle Client, BDE 5.0 с SQL Links к Oracle.
2
Запустите установщик → каталог: Program Files\Sprut.
3
Настройте Windows Registry: номер рабочего места, параметры DB LINK.
4
Настройте Oracle Client (TNS) и BDE для подключения к серверу БД.
👤
Управление пользователями и правами
›
1
Модуль «Администрирование» → «Профайлы» → создайте профайл для каждой роли.
2
Назначьте привилегии: доступные модули, разрешённые действия, лимиты.
3
Создайте пользователя → привяжите к профайлу → выдайте логин и пароль.
⚠️
Принцип минимальных привилегий — давайте только те права которые нужны. Лишние права — источник ошибок и злоупотреблений.
🔍
Мониторинг и аудит
›
1
Журнал «Сессии пользователей» — кто и когда работал в системе.
2
Журнал «Аудит» — все изменения: кто, когда, что изменил.
3
Регулярно очищайте таблицы аудита чтобы БД не разрасталась.
🖥️ Сервер торговой площадки — Глава 30
⚙️
Установка и конфигурирование сервера ТП
›
1
Инсталляция сервера ТП — отдельный установщик.
2
Модуль «Настройки сервера ТП»: Общие параметры, Справочники, Сервис.
3
Настройте подключение к основному серверу Oracle и список торговых площадок.
🚀
Запуск, остановка, мониторинг
›
1
Запуск/остановка: через сервисы Windows или командную строку (/start, /stop).
2
Модуль «Монитор обмена данными» — статус всех обменов в реальном времени.
3
При сбое — проверьте логи в папке сервера ТП, перезапустите службу.
📁 Файловый обмен — Глава 29
🔄
Настройка обмена
›
1
Используется при отсутствии постоянной on-line связи с основным сервером.
2
Варианты: через общедоступный файловый сервер или через почтовый сервер.
3
FServer.ini — основной файл настроек. Укажите пути папок обмена и расписание.
📧
Обмен через почту
›
1
На сервере ТП: укажите SMTP, адрес отправки, адрес получения.
2
Расписание отправки/приёма — каждые 15-30 минут.
3
Ежедневно проверяйте успешность обменов через монитор.
⚠️
Задержка данных более 2 часов — ненормальная ситуация, требует немедленного вмешательства.
📱 Обмен с ТСД — Глава 27
📡
Регистрация и настройка ТСД
›
1
Модуль «Сервер ТСД» → «Регистрация ТСД» → укажите ID устройства.
2
Настройте Wi-Fi на ТСД — должен видеть сервер ТСД.
3
Передача каталога на ТСД: Сервер ТСД → «Передача данных о товарах».
