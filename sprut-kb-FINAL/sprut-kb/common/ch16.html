Глава 16 · Справочное руководство

Учёт материалов и малоценки

Приходование, передача, списание материалов и малоценки. Особенности бухгалтерского учёта.

ОператорБухгалтер
Введение

Материалы и малоценка — товары, которые не предназначены для продажи, а используются внутри компании. Учитываются в ПС «Спрут» через специальные виды накладных.

Раздел 1

Учёт материалов

Организация учёта

В Каталоге товаров создаётся специальная группа «Не товары» — в неё заносятся все материалы (не участвующие в товарообороте).

Приходование материалов

1
Создать приходную накладную → вид «Материалы».
2
При оприходовании система запросит счёт дебета — выбрать из справочника «План бухгалтерских счетов».
3
Провести накладную → материалы поступают на склад.
ℹ️
Счёт дебета оприходования используется при экспорте бухгалтерских данных в 1С. В параметрах экспорта для вида «Материалы» нужно указать только счёт кредита — счёт дебета берётся из накладной.

Передача и списание материалов

Передача между отделами — через передаточные накладные. Списание — через накладную списания. Каждая операция формирует бухгалтерскую проводку при экспорте.

Раздел 2

Учёт малоценки

Малоценка — предметы труда стоимостью ниже лимита (инструменты, спецодежда, канцтовары). Учёт аналогичен материалам, но с дополнительными операциями.

Приходование — вид накладной «Малоценка», указывается счёт дебета.
Передача в отделы — передаточная накладная → малоценка числится за конкретным отделом/сотрудником.
Списание — акт списания → формируется проводка на расходы.
Продажа — возможна продажа материалов и малоценки через заказ, но с особой схемой бухгалтерских проводок.
Раздел 3

Экспорт по типам поставщиков

В ПС «Спрут» поставщики делятся на Внутренних, Импортных и Отечественных. Параметры экспорта бухгалтерских данных настраиваются раздельно для каждого вида поставщика. Это позволяет формировать разные проводки для товаров от разных типов поставщиков.
