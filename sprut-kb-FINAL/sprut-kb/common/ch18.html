Глава 18 · Справочное руководство

Проблемные и ошибочные товары, пересортица

Типы проблемных товаров, дублирование в каталоге, исправление пересортицы, работа с журналом состояния склада.

ОператорТовароведДиректор
Введение

В процессе работы возникают объективные и субъективные ошибки, ведущие к некорректным данным. Их необходимо своевременно исправлять для обеспечения корректности отчётности и информации об остатках.

Раздел 1

Типы проблемных товаров

ТипПричинаПоследствие
Проблемные по продажамПродан товар которого нет на остатках (отрицательный остаток)Искажение финансовой отчётности, некорректная себестоимость
Проблемные по передаточнымОшибка при перемещении между складамиРасхождение остатков на складах
Проблемные по списаниямСписан товар которого не былоОтрицательный остаток, ошибка в учёте
Раздел 2

Дублирование товаров в каталоге

Один из основных источников пересортицы — дублирующие карточки товара с одинаковым или схожим наименованием.

1
Найдите оба дублирующих товара в каталоге → определите «правильный» (с историей).
2
Перенесите все остатки с «неправильного» на «правильный» через корректировочные накладные.
3
Замените штрих-код «неправильного» или установите признак «Устаревший».
⚠️
Никогда не удаляйте товар с историей операций — это нарушит бухгалтерскую отчётность. Только помечайте признаком «Устаревший» — он перестанет отображаться при поиске, но история сохранится.
Раздел 3

Журнал «Состояние склада» и журнал «Проблемных приходов»

Журнал «Состояние склада» — основной инструмент выявления проблем. Показывает текущие остатки по всем товарам. Отрицательный остаток = проблемный товар.

Журнал «Проблемных приходов» — специальный инструмент для отслеживания и систематического решения накопленных проблем.

Раздел 4

Решение пересортицы при продаже

1
Журнал «Состояние склада» → найдите товар с отрицательным остатком.
2
Определите какой реальный товар был продан вместо него (по штрих-коду, внешнему виду, датам).
3
Создайте приходную накладную на проблемный товар (по нулевой цене или себестоимости).
4
Создайте накладную списания реально проданного товара.

Решение пересортицы при оприходовании

1
Найдите приходную накладную с ошибочным товаром.
2
Если накладная ещё в статусе «Готовится» — исправьте напрямую.
3
Если уже проведена → создайте корректировочные накладные: возврат ошибочного товара + приход правильного.

Ошибки с единицами измерения при приходе

Если товар был оприходован в неверной единице (например, кг вместо шт) — создайте накладную списания в неверной единице и приходную накладную в правильной единице на то же количество.

Решение проблемных по результатам инвентаризации

ВариантДействие
Списание по себестоимости + приходование по нулевым ценамСписать проблемный по фактической себестоимости, приходовать правильный по 0
Списание по нулевым ценам + оприходование по списанной себестоимостиОбратная схема — цена переносится на правильный товар
