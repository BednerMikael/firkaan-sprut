Ваши основные модули: «Поставки». Управляете заказами, контролируете взаиморасчёты с поставщиками.
📋 Заказы на поставку — Глава 5
🆕
Создание заказа на поставку
›
1
Модуль «Поставки» → Журнал «Заказов на поставку» → «Добавить».
2
Выберите поставщика и вид: Импортный / Отечественный / Внутренний.
3
Добавьте товары по штрих-коду или из каталога цен поставщика.
4
Укажите количество и цену для каждой позиции → сохраните (статус «Готовится»).
💰
Варианты оплаты поставщику
›
1
Предоплата 100% — оплата до поставки.
2
Частичная предоплата — часть до поставки, остаток после.
3
С отсрочкой платежа — оплата в течение установленного срока.
4
Консигнация — оплата по факту реализации товара.
⚠️
Превышение максимальной задолженности или отсрочки — система заблокирует регистрацию новой оплаты.
📄
Формирование прихода из заказа
›
1
После поставки → откройте заказ → «Сформировать приход».
2
Черновик приходной накладной создан с данными из заказа.
3
Передайте товароведу для проверки фактических количеств и проведения.
4
После приёмки переведите заказ в статус «Выполнен».
💵 Взаиморасчёты с поставщиками — Глава 5
💳
Регистрация оплаты заказа
›
1
Откройте заказ → «Регистрация оплаты» → укажите сумму, дату, форму оплаты.
2
Для отсрочки — система проверит максимальный срок и лимит задолженности.
3
Остаток на счету поставщика — деньги зачтённые от возвратов, используются при следующих оплатах.
📊
Контроль задолженности
›
1
Модуль «Финансы» → Журнал «Состояния счетов поставщиков» → текущий долг по каждому.
2
Просроченные платежи выделяются отдельно.
3
Каталог цен поставщиков — для сравнения цен и переговоров.
↩️ Возвраты поставщикам — Глава 8
📦
Оформление возврата
›
1
Журнал «Заказов на поставку» → «Добавить возврат» → выберите поставщика.
2
Укажите номер приходной накладной для возврата (или режим «по всем приходным»).
3
Добавьте возвращаемые товары из свободных остатков.
4
Статус «Составлен» → задолженность уменьшится на сумму возврата.
⚠️
Возврат только по накладным в статусе «Передано на склад».
📋 Заявки на поставку — Глава 5
📝
Создание заявки и конвертация в заказ
›
1
Журнал «Заявки на поставку» — предварительные заявки до оформления заказа.
2
Формирование из отчёта «Интенсивность продаж» — автоматически по статистике.
3
Импорт товаров в заявку из Excel-файла поставщика.
4
Заявка конвертируется в заказ на поставку одной кнопкой.
