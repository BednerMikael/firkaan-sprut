Глава 27 · Справочное руководство

Организация обмена информацией с ТСД

Регистрация терминалов сбора данных, приложения, приход на ТСД, инвентаризация, сервер ТСД.

IT-администраторТоваровед
Введение

Терминалы сбора данных (ТСД) — мобильные устройства со сканером штрих-кодов. Используются для ускорения приёмки товаров, проведения инвентаризаций и пересчётов без ввода данных вручную с клавиатуры.

Раздел 1

Регистрация и настройка ТСД

1
Модуль «Сервер ТСД» → «Регистрация ТСД» → указать уникальный ID устройства, его тип и модель.
2
Настроить Wi-Fi на ТСД — устройство должно видеть сервер ТСД по сети.
3
Зарегистрировать пользователей ТСД — список сотрудников которые могут работать с терминалом.
4
Установить приложения на ТСД: «Приход» и «Инвентаризация» — отдельные приложения для каждого типа работы.
Раздел 2

Приложения для ТСД

Приход на ТСД

1
Сервер ТСД → «Передача ЗНП (заданий на приход)» → список товаров ожидаемой поставки загружается на ТСД.
2
Сотрудник на складе сканирует штрих-коды поступивших товаров, вводит количества.
3
По окончании — ТСД передаёт данные на сервер → автоматически формируется приходная накладная в состоянии «Готовится».
4
Оператор проверяет и проводит накладную в BackOffice.

Инвентаризация на ТСД

1
Сервер ТСД → «Передача ЗНП (задание на пересчёт)» → список товаров для инвентаризации загружается на ТСД.
2
Сотрудник обходит склад, сканирует товары и вводит фактические количества.
3
Данные передаются в Журнал «Пересчёт товаров» → оператор завершает пересчёт в системе.
ℹ️
Автоматизация инвентаризаций — можно настроить автоматическое формирование заданий на пересчёт по расписанию (например, каждую ночь для определённых групп товаров).
