Возвраты от клиентов

Журнал возвратов, типы возвратных накладных, оформление, возврат сверх отгруженного, тара.

Менеджер по продажамОператорБухгалтер
Введение

Возвраты товаров и тары от клиентов производятся в Журнале «Возвратов от клиентов» модуля «Приход». Журнал предназначен для упрощённой процедуры возвратов, когда товары продавались из разных партий или нужен общий возврат без указания номеров накладных.

Все возвраты через POS поступают в Журнал приходных накладных напрямую и отдельно в Журнал возвратов не попадают.

Раздел 1

Журнал возвратов от клиентов

«Общие» возвраты (введённые вручную) — создаются в Журнале возвратов и порождают «Порождённые» возвратные накладные в Журнале приходных накладных.
По одному общему возврату может генерироваться несколько порождённых накладных (по разным расходным / налоговым).
Экспорт бухгалтерских данных и отчётность формируются из Журнала приходных накладных (порождённые возвраты).

Типы возвратных накладных

ТипПрименение
ТоварСтандартный возврат товара + невозвратной тары (бутылки), связанной с товаром
Возвратная тараВозврат ящиков, кегов, палет
Невозвратная тараБутылки, банки — если учитываются отдельно (оптовая торговля)

Поля возвратной накладной

ПолеОписание
КлиентЮр. лицо или физ. лицо
СкладЕсли не задан — товары возвращаются на те склады, с которых продавались
Тип документаЛюбой документ / По расходной накладной / По налоговой накладной
Форма оплатыНаличная (возврат через кассу) / Безналичная. Важно: наличные возвраты не попадают в Книгу продаж по НДС
Раздел 2

Оформление возврата товара

1
Журнал «Возвратов от клиентов» → «Добавить» → выберите клиента, тип накладной «Товар».
2
Укажите тип документа:
• Любой документ — возврат из всех отгрузок клиенту.
• По расходной / налоговой — возврат из конкретных накладных (автоматически берутся из наиболее поздних отгрузок).
3
Добавьте товары — только из того что было отгружено этому клиенту и не было возвращено ранее.
4
При необходимости — укажите количество сверх отгруженного (если разрешено в настройках системы).
5
Проведите → порождаются приходные накладные и сумма зачисляется на лицевой счёт клиента (или выплачивается наличными).
⚠️
Если форма оплаты «Наличная (возврат через кассу)» — такой возврат не попадает в Книгу продаж модуля «Журналы по НДС». Учтите это при налоговой отчётности.
Раздел 3

Возврат сверх отгруженного клиенту

Клиент может вернуть больше товара, чем ему было отгружено. Разрешение настраивается в Свойствах ПС «Спрут» → «Возвраты от клиента – Разрешить возврат товара больше чем было продано».

Порождённые накладные на излишний товар создаются с типом отправителя «Физическое / Юридическое лицо» и типом документа «Любой документ».
Сумма за излишний возврат зачисляется на остаток на лицевом счету клиента — для последующих взаиморасчётов.
Бухгалтерский счёт для таких операций задаётся в Свойствах ПС «Спрут» → «Бухгалтерские настройки – Проводки и счета».
Раздел 4

Возврат возвратной тары (ящики, кеги)

1
Журнал «Возвратов от клиентов» → «Добавить» → тип «Возвратная тара».
2
Выберите клиента → добавьте тару и количество.
3
Проведите → тара поступает на склад, взаиморасчёты с клиентом по таре закрываются.
ℹ️
При проведении система автоматически возвращает товары на те склады, с которых происходила отгрузка — даже если склад не указан явно в накладной.
