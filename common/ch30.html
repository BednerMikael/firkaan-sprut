Глава 30 · Справочное руководство

Организация работы на Сервере Торговой площадки

Конфигурирование, модули СТП, мониторинг обмена, управление из командной строки.

IT-администратор
Введение

Сервер Торговой площадки (СТП) — приложение с базой данных Interbase, обеспечивающее автономную работу торговой площадки и обмен данными между СПРУТ POS и основной БД Oracle. На каждую торговую площадку — один СТП.

Раздел 1

Конфигурации работы СТП

КонфигурацияОписание
On-line с основным серверомПрямое соединение TCP/IP. Данные передаются в реальном времени.
Файловый обмен через файловый серверДанные через общую папку. Задержка = период синхронизации.
Файловый обмен через почтуДанные через email. Для магазинов только с интернетом.
Раздел 2

Установка и первичная настройка

1
Запустить инсталлятор СТП на компьютере торговой площадки.
2
Модуль «Настройки сервера ТП» → вкладка «Общие параметры»: указать название ТП, ID торговой площадки, параметры подключения к основному серверу.
3
Вкладка «Справочники» — настройка тиражирования: какие справочники получать с основного сервера (каталог товаров, клиенты, ценовые категории).
4
Вкладка «Сервис» — расписание обменов, архивирование БД, очистка логов.
5
Вкладка «Подготовка новой БД» — первоначальная загрузка данных с основного сервера.
Раздел 3

Запуск, остановка, мониторинг

ОперацияСпособ
ЗапускЧерез сервисы Windows / ярлык / командная строка /start
ОстановкаЧерез сервисы Windows / командная строка /stop
ПерезапускКомандная строка /restart

Модуль «Монитор обмена данными» — отображает в реальном времени все активные обмены: статус, время последнего успешного обмена, ошибки.

⚠️
При сбое обмена — проверьте логи в папке сервера ТП, проверьте доступность основного сервера, перезапустите сервис СТП. Если проблема не решена — сообщите в IT-поддержку через OKDesk.
Раздел 4

Модули Сервера торговой площадки

МодульНазначение
Главное меню площадкиИнтерфейс запуска всех модулей СТП
Приход ТППриходование товаров прямо на торговой площадке без обращения к основной БД
Возвраты поставщикам ТПОформление возвратов на торговой площадке
Заказы на поставку ТПСоздание заказов на поставку локально
Кассовая книга ТПРабота с Z-отчётами и кассовыми ордерами на площадке
Акты приёмки по передаточнымПриёмка товаров при внутренних перемещениях
ℹ️
Модули СТП работают с локальной базой Interbase — они функционируют даже при отсутствии связи с основным сервером. При восстановлении связи данные автоматически синхронизируются.
