Приходование товаров

Приходные накладные, типы и виды, проведение, контроль цен, акты приёмки, перемещение между складами.

Введение

Приходование товаров — ввод в ПС «СПРУТ» документов, формирующих остатки на складах. Все документы поступления хранятся в журнале «Приходных накладных» модуля «Приход».

Раздел 1

Учёт поступления товаров на склады

Список складов вводится в Справочнике «Подгрупп, компаний и складов». Для каждой компании ведутся «Договора со складами» — на каких складах компания может хранить свои товары. Товары приходуются только на склады, с которыми у компании есть договор.

ℹ️
Если количество и ассортимент в инвойсе фактически не совпадают с поступившим — составляйте Акт приёмки. Акты приёмки возможны только для Обычных приходных накладных.

Виды накладных

Каждый тип может иметь вид: обычная, специальная_1, специальная_2, брак, тара, материалы, малоценка, по акту приёмки.

Порядок создания накладной

1
Модуль «Приход» → Журнал «Приходных накладных» → «Добавить».
2
Заполните шапку: Тип, Поставщик, Склад, реквизиты документа поставщика.
3
Табличная часть: добавьте товары по штрих-коду или поиску → укажите количество и приходную цену (без НДС).
4
Система покажет последнюю приходную цену и текущую цену реализации для сравнения.
5
Сохраните → статус «Готовится».
Раздел 3

Проведение накладной — оприходование на склад

1
Проверьте все данные: товары, количества, цены, реквизиты документа поставщика.
2
Система проверит контроль цен: отклонение приходной цены от предыдущей (настраиваемый %).
3
При необходимости — выполните расценку (расчёт цен реализации) перед проведением.
4
Нажмите «Провести» → статус «Передано на склад» → товары зачислены на остатки.
5
После проведения — печать ценников и этикеток прямо из накладной.
⚠️
После проведения отменить накладную нельзя без специальной процедуры (требует прав администратора). Тщательно проверяйте данные до нажатия «Провести».

Контроль цен при оприходовании

Контроль отклонения наценки — если рассчитанная наценка отклоняется от плановой сверх допустимого %, система предупреждает или блокирует проведение.
Блокировка при цене прихода выше продажной — для заданных торговых площадок нельзя оприходовать товар с приходной ценой выше текущей цены реализации.
Контроль ВМД — для импортных накладных проверяется наличие и соответствие ВМД.
Раздел 4

Журнал актов приёмки товаров на склад

Акт приёмки составляется когда фактически поступившие товары отличаются от накладной поставщика по ассортименту или количеству. Акт составляется после оприходования основной накладной.

1
Журнал «Актов приёмки» → «Добавить» → выберите оприходованную накладную → список товаров копируется автоматически.
2
Отредактируйте фактические количества по каждой позиции. Добавьте товары обнаруженные сверх накладной (количество по накладной = 0).
3
Нажмите «Сформировать» → статус «Составлен». После этого редактирование запрещено.
4
Система автоматически создаст:
• Доп. приходную накладную — на товары обнаруженные сверх накладной.
• Расходную накладную (возврат) — на товары недостающие по накладной.
5
Оприходуйте доп. накладную — товары появятся на складе.
ℹ️
Для ускорения ввода фактических данных — экспортируйте акт в Excel (кнопка «Экспорт в Excel»), заполните колонку «Фактическое количество» и импортируйте обратно.
Раздел 5

Перемещение товаров между складами

1
Создайте приходную накладную типа «По передаточной».
2
Укажите склад-источник (в поле «Отправитель») и склад-получатель.
3
Добавьте товары с количествами → проведите накладную.
Производные товары (ящики, упаковки) при перемещении с оптовой базы в магазин автоматически конвертируются в базовые (штучные).
Импорт передаточной накладной — достаточно указать номер расходного документа внутреннего поставщика.

Отмена проведения накладной

Доступна только для накладных в статусе «Передано на склад» при наличии прав. Условия отмены: остатки по товарам не меньше количества в накладной (нельзя отменить если товар уже продан).
